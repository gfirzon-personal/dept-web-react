## Run the app in dev mode locally
```sh
npm run dev
```

## To install globally, you would need to use:
```sh
npm install -g <package-name>
```

## To see detailed information about the vulnerabilities without fixing them:
```sh
npm audit
```

## This will automatically update packages to patched versions that fix the vulnerabilities.
```sh
npm audit fix
```
