Contextual colors:

text-primary - Blue
text-secondary - Gray
text-success - Green
text-danger - Red
text-warning - Yellow/Orange
text-info - Cyan
text-light - Light gray
text-dark - Dark gray/black
Additional text utilities:

text-muted - Muted/dimmed text
text-white - White
text-body - Default body color
text-black-50 - Black with 50% opacity
text-white-50 - White with 50% opacity
You can also combine with other classes like:

fw-bold - Bold text
fst-italic - Italic text
text-decoration-underline - Underlined text
Or leave it empty '' for default styling.
